# New Drone Frontend

A new React frontend for the Drone Model API.

## Features

- List drones
- View drone details
- Connects to backend API at http://localhost:3000

## Setup

1. Install dependencies: 
pm install
2. Start the dev server: 
pm run dev

## Backend

The backend files are included in the ackend/ directory. To run the backend:

1. cd backend
2. npm install
3. npm run dev

Make sure MongoDB is running.
